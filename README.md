About branch:
-------------
This branch is intended for install TKZ application in Ubuntu-22.04.

Note:
-----
Before installation don't forget to check SHA256 hash, and GPG signature. Instruction how to do it you find in "main" branch. If you have any qustions ask in <jbospc.tkz@gmail.com> email.

Installation:
-------------
To install application in your OS download "tkz_8.22-1_all.deb" and use "sudo dpkg {package name}" to install it.
